🚚 Dashboard de Monitoramento Logístico

Este projeto foi desenvolvido para o Desafio de Dados, com o objetivo de criar um dashboard interativo para análise de entregas logísticas no Brasil.

📊 Objetivo

O sistema permite:

Monitorar entregas atrasadas
Identificar transportadoras com pior desempenho
Analisar atrasos por região
Priorizar entregas críticas
Apoiar decisões operacionais em tempo real
🧠 Funcionalidades do Dashboard
KPIs (Total de entregas, atrasos, percentual, etc.)
Filtros interativos (região, transportadora, status)
Ranking de entregas mais críticas
Análise de atrasos por região
Comparação entre transportadoras
Destaque visual de atrasos
🗂️ Base de Dados

A base utilizada é fictícia e contém:

ID da entrega
Transportadora
Região
Prazo previsto
Tempo real de entrega
⚙️ Tecnologias utilizadas
Python
Streamlit
Pandas
Plotly
